## :computer: Getting Started

---
# Clone the repo
```
git clone https://github.com/Sravanikonapalli/my-personal-protofilo.git
```

# Navigate into the project folder
```
cd protofilo
```
# Install dependencies
```
npm install
```

# Start the development server
```
npm start
```
